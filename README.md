## :fire: Instalação

**Clonando o repositório:**

```
$ git clone https://github.com/AfonsoMachado/proffy-nlw.git
```

**Executando back-end**

```bash
# Go to server folder
$ cd proffy-server

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Executando em http://localhost:3333/
